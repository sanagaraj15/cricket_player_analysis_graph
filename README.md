📌 Project Overview

This project focuses on analyzing cricket player performance using data visualization techniques. The objective is to evaluate player statistics such as runs, wickets, and match performance using multiple graphical representations.

The project demonstrates how different visualization techniques can be used to analyze sports performance data effectively. Various chart types such as line graphs, bar charts, pie charts, histograms, area charts, step charts, box plots, and heatmaps are used to visualize and compare player statistics.

The system creates a multi-plot dashboard using Matplotlib subplots, allowing multiple visual analytics to be displayed in a single figure window.

🎯 Objectives

Analyze cricket player performance statistics

Visualize runs, wickets, and matches played

Compare player statistics using multiple chart types

Understand data distribution using statistical graphs

Create an advanced visualization dashboard using Matplotlib

Demonstrate subplot-based data visualization techniques in Python

📊 Dashboard Visualizations

The dashboard contains 9 different data visualizations.

Player Performance Charts

Player Runs Performance (Line Graph)
Shows run scoring trends across players.

Player Wickets Comparison (Bar Chart)
Compares wicket performance between players.

Matches Played (Bar Chart)
Displays the number of matches played by each player.

Runs Distribution by Player (Pie Chart)
Shows percentage contribution of each player's runs.

Statistical Analysis Charts

Runs Distribution (Histogram)
Analyzes the statistical distribution of runs.

Runs Area Chart
Represents cumulative run performance.

Wickets Step Chart
Shows wicket progression across players.

Runs Box Plot
Displays statistical spread including median and quartiles.

Performance Analysis

Player Performance Heatmap

Visualizes match-wise player performance using color intensity.

🖥 How to Run the Project

1️⃣ Clone the repository

git clone https://github.com/sanagaraj15/cricket_player_analysis.git

2️⃣ Navigate to the project folder

cd cricket_player_analysis

3️⃣ Install required libraries

pip install matplotlib numpy

4️⃣ Run the project
python cricketgraph.py

The program will generate a Cricket Player Performance Analysis Dashboard with multiple visualizations displayed in a single figure.
